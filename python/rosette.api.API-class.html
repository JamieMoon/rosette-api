Class API

source code

Rosette Python Client Binding API; representation of a Rosette server. Call instance methods upon this object to obtain Operator objects which can communicate with particular Rosette server endpoints.

Parameters:
  • user_key - (Optional; required for servers requiring authentication.) An authentication string to be sent as user_key with all requests. The default Rosette server requires authentication. to the server.
  • service_url - (Optional) The root URL (string) of the Rosette service to which this API object will be bound. The default is that of Basis Technology's public Rosette server.

morphology(self, facet='complete')

source code 

Create an Operator to morphologically analyze a text. Produce an operator which returns a specific facet of the morphological analyses of texts to which it is applied. MorphologyOutput.COMPLETE (the default) requests all available facets.

Parameters:
  • facet (An element of MorphologyOutput.) - The facet desired, to be returned by the created Operator.

entities(self, linked)

source code 

Create an Operator to identify named entities found in the texts to which it is applied. Linked entity information is optional, and its need must be specified at the time the operator is created.

Parameters:
  • linked (Boolean) - Specifies whether or not linked entity information will be wanted.
